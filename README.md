## Run Locally

Clone the project

bash
  git clone https://github.com/cris96spa/portfolio.git


Go to the project directory

bash
  cd portfolio


MAKE  Migration

```bash
  python manage.py makemigrations
```

MAKE  Migration

```bash
  python manage.py migrate     
```
Start the server

```bash
  python manage.py runserver     
```
